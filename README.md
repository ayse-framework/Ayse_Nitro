# Ayse_Nitro

### Create this item:

```lua
["nos"] = {
    label = "NOS Bottle",
    weight = 2000,
    stack = false,
    close = true,
    client = {
        disable = { move = true, car = true, combat = true },
        usetime = 3500,
        cancel = false,
        export = "Ayse_Nitro.nos"
    }
},
```

## Dependencies:
* [Ayse Core](https://github.com/ayse-framework/Ayse_Core)
* [ox inventory](https://github.com/overextended/ox_inventory/releases)
